Complexity Analysis

Two main things to measure usefulness of the algorithm

  • Time Complexity
  • Space Complexity

If we have given input to the computer, Execution Time is the time it took to execute & return the desired output

# Input -> Execution Time
I1 -> T1
I2 -> T2
I3 -> T3
    .
    .
    .
In -> Tn

If we the plot points in the co-ordinate system by taking input(I1, I2…) on X-Axis & execution time(T1, T2…) on the Y-Axis, we can derive a curve by joining all the points plotted(also known as curve fitting). we can then derive a function out of the curve which will be like

f(I) -> T

Example:

# Let's assume the function of the graph
f(i) = 2i^3 + 4i^2 + 8i + 1

# Since this alalysis is mainly considered when dealt with large inputs, in those cases constants can be neglected
f(i) = 2i^3 + 4i^2 + 8i

# To simplify it further, we consider the term with largest degree & neglecting the lesser degree one's
f(i) = 2i^3

# To simplify it even further, we will remove the constants
f(i) = i^3

Since it is not the exact function we have derived earlier from the graph, we call it as Order which is Time Complexity
O(i^3)

# Possible big O notation values
O(1)
O(n)
O(n^2)
O(log(n))
O(nlog(n))

Space Complixity will be calculated in the same as time complexity but rather than exection time it the space required to execute
