Python How it works:

Low Level Language:
Source code ->[compiling] machine readable code
Platform dependent

High Level Language:
Source code ->[compiling] Intermediate code ->[Interpreter] Machine readable code
Interpreter - PVM(python virtual machine)
Platform independent

  1. Compiling source code to bytecode
  2. Interpret the bytecode in python virtual machine

Python compiler:
Cpython in. c, pypy in python, jython in java, ironpython in .net
Converts .py file to .pyc file (Bytecode, platform independent & low level representation)
