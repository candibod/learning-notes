C, C++, and Java are examples of static languages. They require you to specify some low-level details like data types for the computer.

int language = 2;

In contrast, dynamic languages (also called scripting languages) do not force you to declare variable types before using them.

Ex: Perl, Ruby, PHP, GO(golang), Python

language = 2
